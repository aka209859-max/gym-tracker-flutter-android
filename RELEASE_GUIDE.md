## 🔧 **リリースビルド作成手順**

### **1. App Bundle（AAB）作成（推奨）**

Google Playの最新要件に対応したフォーマット。

```bash
cd /home/user/flutter_app

# AABビルド作成
flutter build appbundle --release

# 生成場所
# build/app/outputs/bundle/release/app-release.aab
```

**ファイルサイズ目安**: 約15〜30MB

---

### **2. APKビルド作成（代替手段）**

直接インストール可能なフォーマット。

```bash
cd /home/user/flutter_app

# APKビルド作成
flutter build apk --release

# 生成場所
# build/app/outputs/flutter-apk/app-release.apk
```

**ファイルサイズ目安**: 約30〜50MB

---

### **3. ビルド後の確認事項**

```bash
# ファイルサイズ確認
ls -lh build/app/outputs/bundle/release/app-release.aab
ls -lh build/app/outputs/flutter-apk/app-release.apk

# 署名確認
keytool -printcert -jarfile build/app/outputs/flutter-apk/app-release.apk

# バージョン確認
# pubspec.yaml の version: 1.0.0+1
```

## 📦 **Google Play Console アップロード手順**

### **Step 1: アカウント登録**

1. **Google Play Console にアクセス**
   - URL: https://play.google.com/console
   - Googleアカウントでログイン

2. **$25 登録料の支払い**
   - クレジットカード/デビットカード必要
   - 1回限りの支払い

3. **開発者プロフィール設定**
   ```
   開発者名: 株式会社Enable
   メールアドレス: [CEOのメールアドレス]
   ウェブサイト: [任意]
   ```

---

### **Step 2: 新しいアプリを作成**

1. **「アプリを作成」ボタンをクリック**

2. **基本情報を入力**
   ```
   アプリ名: GYM MATCH
   デフォルト言語: 日本語（日本）
   アプリまたはゲーム: アプリ
   無料または有料: 無料
   ```

3. **アプリケーションID確認**
   ```
   jp.nexa.fitsync
   ```

---

### **Step 3: ストア掲載情報の入力**

#### **必須項目**

1. **アプリ名**
   ```
   GYM MATCH - 科学的トレーニング管理
   ```

2. **簡単な説明**（80文字以内）
   ```
   40本の論文データに基づく科学的トレーニング記録・分析アプリ
   ```

3. **詳細な説明**（4,000文字以内）
   ```
   【GYM MATCHとは】
   科学的根拠に基づいたトレーニング管理アプリです。40本以上の研究論文データを活用し、あなたのトレーニングを最適化します。
   
   【主な機能】
   ✅ トレーニング記録管理
   ✅ AI科学的コーチング（40本の論文データ活用）
   ✅ 成長予測分析
   ✅ 効果分析レポート
   ✅ テンプレート機能
   ✅ ワークアウト履歴追跡
   
   【対象ユーザー】
   - 科学的にトレーニングを管理したい方
   - パーソナルトレーニング効果を最大化したい方
   - データに基づいた成長を実感したい方
   
   【開発元】
   株式会社Enable - AI戦略とデータサイエンスを活用した革新的なフィットネスソリューション
   ```

4. **スクリーンショット（必須）**
   - 携帯電話用: 最低2枚、推奨8枚
   - サイズ: 320px〜3840px（縦横比 16:9 または 9:16）
   - フォーマット: PNG または JPEG

5. **アイコン（必須）**
   - サイズ: 512x512px
   - フォーマット: PNG（透過背景、32bit）
   - 既存のアプリアイコンを使用可能

6. **フィーチャーグラフィック（必須）**
   - サイズ: 1024x500px
   - フォーマット: PNG または JPEG

### **Step 6: AAB/APKのアップロード**

4. **リリース名とリリースノートを入力**
   ```
   リリース名: 1.0.0
   
   リリースノート（日本語）:
   初回リリース - GYM MATCH v1.0.0
   
   【新機能】
   ✅ トレーニング記録管理
   ✅ AI科学的コーチング（40本の論文データ）
   ✅ 成長予測分析
   ✅ 効果分析レポート
   ✅ テンプレート機能
   ```

5. **「保存」→「確認」→「本番環境にロールアウト」**

### **問題2: 署名エラー**

**原因**: key.properties の設定ミス

**解決策**:
```bash
# key.properties を確認
cat android/key.properties

# 内容が正しいか確認
storePassword=gymmatch2025
keyPassword=gymmatch2025
keyAlias=release
storeFile=release-key.jks
```
