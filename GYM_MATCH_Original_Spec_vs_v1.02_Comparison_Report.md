## エグゼクティブサマリー

### 主要な変更点

v1.02は、**元仕様書から大幅に機能を絞り込んだMVP（Minimum Viable Product）版**として実装されました。

#### 変更の方向性
```
元仕様書: 総合フィットネスアプリ（7大機能）
     ↓ 機能絞り込み・MVP化
v1.02: トレーニング記録 + AI分析 + パートナー機能（3大機能）
```

#### 変更統計

| カテゴリ | 元仕様書 | v1.02実装 | 差分 |
|---------|---------|----------|------|
| **主要機能数** | 7機能 | 3機能 | -4機能 |
| **AI利用回数（Free）** | 広告視聴で月3回 | 3回/月（広告不要）→ v1.02で広告追加 | 変更 |
| **AI利用回数（Premium）** | 10回/月 | 20回/月 | +10回 |
| **AI利用回数（Pro）** | 30回/月 | 無制限（∞） | 大幅改善 |
| **ジム検索機能** | ✅ 実装予定 | ❌ 未実装 | 削除 |
| **混雑度表示** | ✅ 実装予定 | ❌ 未実装 | 削除 |
| **計算ツール** | ✅ 実装予定 | ❌ 未実装 | 削除 |
| **目標・達成管理** | ✅ 実装予定 | ❌ 未実装 | 削除 |
| **Firestoreコレクション数** | 22 | 約10-12 | -10コレクション |

## AI機能の重大な変更

### 1. AI利用回数の変更

#### 元仕様書（計画）
```yaml
無料プラン:
  - リワード広告視聴必須
  - 月3回まで
  - 広告視聴ごとに+1回付与

Premium プラン:
  - 月10回

Pro プラン:
  - 月30回
```

#### v1.02実装
```yaml
無料プラン（v1.01）:
  - 月3回（広告視聴不要）
  - リワード広告視聴で+1回追加可能
  
無料プラン（v1.02）:
  - 月3回（広告視聴不要）
  - リワード広告実装（+1回/視聴）✨ NEW

Premium プラン:
  - 月20回（+10回増量）

Pro プラン:
  - 無制限（∞表示）（大幅改善）
```

### 変更理由と影響

#### 1.1 無料プランへのリワード動画広告追加（v1.02）
```
理由:
  1. 収益化強化（月間+¥50k〜¥100k見込み）
  2. 無料ユーザーへのAI利用機会拡張
  3. 元仕様書の設計に近づける

UX設計:
  - ユーザー主導型（押しつけがましくない）
  - AI残回数が0のときのみ「動画を見て1回分ゲット」ボタン表示
  - 30秒視聴 → 即座に+1回付与
```

#### 1.2 Premium AIプラン増量（10回 → 20回）
```
理由:
  1. Premiumプランの価値向上
  2. Free 3回 → Premium 20回（6.7倍）の明確な価値提案
  3. Pro CVRへの影響軽減（20回でも不足 → Proへ誘導）

ビジネスインパクト:
  - Premium CVR予測: +0.5%
  - 価値認識向上: +15%
```

#### 1.3 Pro プラン無制限化（30回 → ∞）
```
理由:
  1. 「無制限」の価値を明確化
  2. "30回"は技術的制限を感じさせる
  3. Pro CVR向上（+0.5%見込み）

実装:
  - バックエンド: `999`を無制限として扱う
  - フロントエンド: `∞`表示
```

### 2. AI機能の内容変更

#### 元仕様書（計画）
```
AI機能（3種類）:
  1. AIトレーニングコーチング
     - 部位別メニュー生成
     - レベル別カスタマイズ（初心者/中級/上級）
     - メニュー保存・履歴管理
     
  2. AI成長予測・効果分析（Premium以上）
     - 1RM予測
     - 成長曲線予測
     - 週次レポート自動生成
     - 疲労管理システム
     
  3. 計算ツール（全プラン共通）
     - RM計算機
     - 体組成計算機
     - カロリー計算機
```

#### v1.02実装
```
AI機能（2種類）:
  1. AIコーチング
     - トレーニング提案
     - フォーム改善アドバイス
     
  2. 成長予測・効果分析
     - 筋力・体重推移予測
     - トレーニング効果の科学的評価
     
疲労管理システム（無料・無制限）:
  ✅ Phase 2a: sRPE + Training Load計算
  ✅ Phase 2b: Personal Factor Multipliers
  ✅ Phase 2c: ACWR + EWMA
  → ローカル計算のみ（APIコスト不要）
  → 全プラン無料・無制限化

削除:
  ❌ レベル別カスタマイズ（初心者/中級/上級）
  ❌ 計算ツール（RM・体組成・カロリー）
```

### 変更理由
```
1. MVP化: コア機能に集中
2. 疲労管理無料化: APIコスト削減、ユーザー価値向上
3. 計算ツール削除: 優先度低、他アプリで代替可能
```

## 実装範囲の縮小理由

### 1. MVP（Minimum Viable Product）戦略

#### 理由
```
1. 市場検証の迅速化
   - コア機能（トレーニング記録 + AI）に集中
   - ユーザーフィードバックを早期収集
   - 市場ニーズを検証

2. 開発コスト削減
   - Google Places/Maps APIコスト削減（月間数万円）
   - 開発工数削減（6-12ヶ月 → 3-6ヶ月）
   - 運用コスト削減

3. 技術的リスク軽減
   - 混雑度データ収集の困難さ（初期ユーザー少数）
   - 複雑な機能の品質担保
```

| パートナー機能 | 🟡 Medium | 🟡 Medium | ✅ |
| 疲労管理 | 🟡 Medium | 🟢 Low | ✅ |
| ジム検索 | 🟢 Low | 🔴 High | ❌ |
| 混雑度表示 | 🟢 Low | 🔴 High | ❌ |
| 計算ツール | 🔵 Very Low | 🟢 Low | ❌ |
| 目標・バッジ | 🔵 Very Low | 🟡 Medium | ❌ |
