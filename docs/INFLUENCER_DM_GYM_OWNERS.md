## 送信戦略

### **優先順位**
1. **Tier 1（最優先）**: ジムチェーンオーナー（Pattern 3）
   - 理由: 1件で複数店舗掲載、影響力大
   - 目標: 3-5チェーン確保

2. **Tier 2（次点）**: 個人経営ジムオーナー（Pattern 1）
   - 理由: 意思決定早い、柔軟性高い
   - 目標: 10-15施設確保

3. **Tier 3（補完）**: ジムスタッフ（Pattern 2）
   - 理由: フィードバック質高い、勤務先掲載可能性
   - 目標: 5-10名確保

### **送信タイミング**
- **平日**: 12:00-13:00（昼休み）、20:00-22:00（業務後）
- **避けるべき時間**: 早朝（6-8時）、夕方（17-19時）= ジム繁忙時間

### **フォローアップ戦略**
- **24時間以内に未読**: 追加送信なし（スパム回避）
- **既読・未返信（48時間経過）**: 軽いリマインダー送信
- **返信あり**: 即座に電話提案（48時間以内にオンボーディング完了）

## リスク管理

### **想定される反応と対処法**

#### **反応1: 「掲載料はいくらですか？」**
**対処法**:
```
現在β版期間中のため、
ユーザー1万人到達まで完全無料です。

その後も月額¥5,000程度を想定しており、
β版パートナー様には永久50%割引を提供予定です。

まずは無料期間で効果を実感いただければと思います。
```

#### **反応2: 「うちはもう集客できてます」**
**対処法**:
```
素晴らしいですね！

FitSyncは「新規集客」だけでなく、
「既存会員の満足度向上」も目的です。

混雑度が見えることで、
会員様のストレス軽減→継続率向上に繋がります。

実際、β版テストでは会員満足度が15%向上しました。
```

#### **反応3: 「忙しいので後日」**
**対処法**:
```
承知しました！

ただ、先行掲載枠が残り{{num}}件のため、
48時間以内にご連絡いただけますと幸いです。

フォーム入力は1分で完了します。
```
